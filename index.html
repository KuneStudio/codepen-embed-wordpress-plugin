CodePen Embed

WordPress Plugin

The Codepen Embed WordPress Plugin is a plugin that will allow you to easily embed CodePen Pens into you WordPress site.

How to Install

  1. Unzip the downloaded Zip File
  2. Upload the **codepen-embed** folder to your WordPress **plugins**'directory
  3. Activate the plugin in the Admin Panel

How to Use This Plugin

In the WISIWYG editor the will be a new CodePen icon.

Click on the CodePen Icon and a popup will appear.

Fill out the options and click 'insert'.

The new shortcode will now appear in the WISIWYG editor.

Options

The CodePen Enbed Plugin has the following fields that need to be filled out to embed your CodePen Pen for use on your WordPress website.

Username This is the Username for the CodePen author of the Pen you want to embed.
URL URL is the last part of you pen's URL. If your pen's URL is http://codepen.io/username/pen/aBcdE. 'aBcdE' is what you put in the Href field.
Display Onload Display Onload is what section of the pen you want to be displayed when it is loaded on your WordPress website.
